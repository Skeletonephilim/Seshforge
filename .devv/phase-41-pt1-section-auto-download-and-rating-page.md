# Phase 41: PT1 Section Auto-Download & Rating Page

## Overview
Implemented automatic report download when PT1 exam sections are completed/skipped, and created a comprehensive Rating & Review page that shows all section evaluations with downloadable individual reports at exam completion.

## User Requirements
1. **Auto-download report** when any PT1 section is completed or skipped
2. **Landing page after exam**: Rating/Review page showing all sections with:
   - Section scores
   - Evaluation summaries
   - Individual download buttons for each section
   - Final comprehensive report download

**Trigger**: When `handleSectionSwitch()` completes a section

### 4. Download Helper Utility

**File**: `src/lib/download-helpers.ts` (NEW)

```typescript
export function downloadMarkdownFile(content: string, filename: string) {
  const blob = new Blob([content], { type: 'text/markdown' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = filename;
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);
  URL.revokeObjectURL(url);
}

export function generateSectionFilename(section: PT1Section): string {
  const sectionNames = {
    web_application: 'Web-Application-Testing',
    network_security: 'Network-Security-Testing',
    active_directory: 'Active-Directory-Testing',
  };
  return `PT1-${sectionNames[section]}-${Date.now()}.md`;
}
```
